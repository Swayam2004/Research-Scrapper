Project Overview

This search engine is designed to help researchers explore academic papers related to clean cookstoves and sustainable energy solutions. It uses BM25 ranking algorithm to provide relevant search results.

Features

  • Fast full-text search across research papers
  • BM25 ranking algorithm for accurate results
  • PDF and Markdown processing capabilities
  • Research paper organization tools

Technologies Used

  • FastAPI for backend API
  • Python for text processing
  • BM25 for search ranking
  • HTML/CSS for frontend
